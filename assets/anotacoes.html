dicas de atalhos do vs code

  • para estrutura basica do arquivo html colocar ! e enter ou tab
  • para editar varias tags iguais ao mesmo tempo, selecionar a primeira tag q quer mudar, apertar command d que vai selecioanar a proxima, seguir apertando command d ate selecionar todos, depois disso com as teclas de para cima e para baixo vc posiciona o cursor onde quer alterar e altera todas de uma vez so
  • para criar uma tag dentro de outra pelo teclado mais rapido, por exemplo, uma imagem e um nav dentro da header, vc pode escrever assim: header>img+nav e ele ja vai criar o header, img, abrir e fechar e o nav e fechar o header. (para criar filhos, usar o > para criar irmaos usar o +
  • se eu quiser dentro do nav criar 5 links, (tag a) posso escrever: header>img+nav>a*5
  • para ja criar a tag com algum atributo basta colocar o atributo entre [ ] e para ja colocar o conteudo pode colocar entre chaves { } por exemplo, a[lang=en]{Skills}
  • Comandos do git que preciso lembrar: - setar a origem do github: git remote add origin https://... - iniciar o git: git init - adicionar um arquivo no git: git add +nome do arquivo ou mais . para adicionar todos da pasta - commitar no git: git commit -m "..."" -- e publicar: git push (ATENCAO, para a primeira vez que for fazer o push, tem que fazer o seguinte comando: git push -u origin master) - para baixar um codigo: git clone https://o endereco copiado do github - para excluir uma pasta: rm -f (se tiver bloqueado e vc tiver mta certeza, usar o -fr)
  • utilidades do terminal: ls para listar, cd para mudar de pasta, clear para limpar o terminal, para copiar todos os arquivos de uma pasta: cp -r pasta/* para onde quero mandar/.
